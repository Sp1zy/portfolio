# Personal Developer Portfolio

A sleek, modern, and high-performance single-page developer portfolio designed to showcase front-end engineering projects, technical skills, and professional contact channels.

## 🚀 Live Demo
You can view the live deployed website here: https://sp1zy.github.io/portfolio/

## ✨ Features
- **Tight & Optimized Spacing:** Features custom, compact padding and section margins optimized for immediate content visibility upon page load.
- **Dynamic Project Showcase:** Implements a dynamic grid system with fully interactive project preview cards.
- **Interactive Component Layout:** Project cards are wrapped in functional semantic anchor elements, disabling raw browser underlines and linking behaviors. Hovering smoothly transforms and lifts the entire card container.
- **Fully Responsive:** Smooth typography scale and flexible padding shifts that fluidly adapt across desktop, tablet, and mobile dimensions.

## 🛠️ Tech Stack
- HTML5 (Structural layout)
- CSS3 (Variables, Layout Grid, Component styling, Transitions)
